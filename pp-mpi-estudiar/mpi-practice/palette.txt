# next / prev (en ejercicio Anillo):
# - `next`: índice del proceso al que se envía el siguiente mensaje. Se calcula como
#   `(rank + 1) % nprocs`, de forma que el último proceso envía al proceso 0.
# - `prev`: índice del proceso del que se recibe el mensaje anterior. Se calcula como
#   `(rank - 1 + nprocs) % nprocs`, evitando negativos y cerrando el anillo.
# Uso: `prev/next` permiten recorrer todos los procesos en un anillo con `MPI_Send`/`MPI_Recv` simples.
